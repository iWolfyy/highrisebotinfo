General Commands

  • !help: Show all commands and help info.
  • !emotes: List all available emotes.
  • !feedback: Send feedback to the bot maker (@Mr_Wolfy).
  • !vip: Check your VIP status in the room.
  • !viplist: List all VIPs in the room.
  • !mod: List all moderator commands.
  • !other: List other miscellaneous commands.
  • !ask: Chat with the bot (ask questions).

Movement Commands

  • !tp @user: Teleport to another user.
  • !pos @user: Get position of a user.
  • !summon @user: Summon another user to your position.
  • reset: Respawn in default position.

Outfit Commands

  • !equip <item> [index]: Equip an item for the bot.
  • !unequip <category>: Unequip an item category for the bot.
  • !change <category> <color>: Change color of outfit category for the bot.

Emote Commands

  • Use the !emote function to get all the emotes and type any emote from the list to use the emote (e.g., rest, zombie).
  • stop: Stop current emote.

Fun & Info Commands

  • !joke: Get a random joke from the bot.
  • !tax: Highrise Gold Tax Calculator from the bot.
  • !news: Show top news of the day from the bot.
  • !fact: Get a random fact from the bot.

Moderator Commands

  • !kick @user: Kick a user.
  • !ban @user: Ban a user.
  • !mute @user: Mute a user.
  • !unban @user: Unban a user.
  • !test @user: Check privileges of a user.
  • !bal: Show bot's gold balance.
  • !tip <amount>: Tip gold to all.
  • !givevip @user: Grant VIP status to a user(Owner only).
  • !removevip @user: Remove VIP status from a user(Owner only).
  • !movebot: Move the bot position permenantly.
  • !maketele [name] [roles...]: Create teleport point.
  • !deltele [name]: Delete teleport point.
  • !teleports: List all teleport points.
